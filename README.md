# Trips React App

A simple React web app that displays a list of trips from a local json as cards. Features include search, sorting by rating, and a modal with more details.

---

## Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)

---

## How to run

1. **Clone the repository**

```bash
git clone https://github.com/d-todorov/react.git
cd my-react-app
```

2. **Run with npm**

```bash
npm install
npm run dev
```
